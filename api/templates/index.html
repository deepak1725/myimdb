Q. What is MyIMDB?

MyIMDB is a collection of Rest'ful API endpoints, build to fetch movie details.

Q. Is there any access restrictions?

We have two roles, Admin and User and each role has certain limitations and privileges

Q. What is the Technology deployed?

We are using Django at Backend accompanied with Django Rest Framework for API endpoints.
We are hosted on Heroku. All communications are in JSON. The project is Open Source, available here

Q. Can i access data from my desktop?

Yes! You can. We have a POSTMAN endpoint collection here, https://www.getpostman.com/collections/2a6083082af6539b297e
You can use this link to import collections right in your POSTMAN desktop client.

Anonymous/Users Privileges

  • Can Fetch Movie Records

  • Can Search records based on Movie name or director name.

  • Can jump pages

Admin Privileges

  • All user associated privileges

  • Can Add/Edit/Delete Movie Records

  • Can Add/Remove Users from Admin Panel

  • Can also View/Edit/Delete Movie/Genre records from Admin Panel
